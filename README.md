# FiveM-Api
C'est une api pour intéragir avec le petite api FiveM

**Pour l'installer :**
```
npm i fivem-api
```
**Pour l'utiliser :**
Ex, pour récupérer le nombre de joueurs sur un serveur : 
```javascript
const fivem = require("fivem-api");

```
